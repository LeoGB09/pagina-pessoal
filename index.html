Um pouco sobre meu trabalho e habilidades

Como desenvolvedor Front-end, utilizo as seguintes tecnologias:

CSS (Cascading Style Sheets): É a linguagem que define o estilo visual de uma página web. É como a roupa que vestimos.

HTML (HyperText Markup Language): É a linguagem que define a estrutura de uma página web. É como o esqueleto de um corpo humano.

JavaScript: É uma linguagem de programação que adiciona interatividade a uma página web. É como o cérebro que controla as ações.
